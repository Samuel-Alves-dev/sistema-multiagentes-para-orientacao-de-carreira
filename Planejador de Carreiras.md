# 💬 ROTEIRO DE ENTREVISTAS

PERGUNTA 1: "Qual a diferença entre aprendizado supervisionado e não supervisionado?"

COMO RESPONDER: Explique que supervisionado usa dados rotulados, enquanto não supervisionado busca padrões sem rótulos.

PERGUNTA 2: "O que é overfitting?"

COMO RESPONDER: É quando o modelo aprende demais os dados de treino e perde capacidade de generalização.

PERGUNTA 3: "Como você trataria dados faltantes?"

COMO RESPONDER: Citar técnicas como remoção, imputação média/mediana ou modelos específicos.

PERGUNTA 4: "Explique a diferença entre Pandas e NumPy."

COMO RESPONDER: NumPy é para arrays numéricos, Pandas é para manipulação de tabelas (DataFrames).

PERGUNTA 5: "Como você comunicaria resultados para alguém não técnico?"

COMO RESPONDER: Usaria visualizações simples, dashboards e linguagem acessível.
